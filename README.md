# Expense Tracker

A simple Expense Tracker web application built using HTML, CSS, JavaScript, and jQuery.  
This project helps users track their income and expenses and view monthly data.

## Features
- Add income and expense transactions
- View total balance, income, and expenses
- Sidebar navigation with responsive layout
- Monthly expense and income data stored using localStorage
- Dynamic UI updates without page reload

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- jQuery
- Browser localStorage
